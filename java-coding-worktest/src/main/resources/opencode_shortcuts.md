# OpenCode 快捷键说明

## 基本操作

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 / 到达底部发送 |
| `Shift` + `Enter` | **输入完不发送** / 用于多行输入 / 直接发送（终端不发送修改键） |
| `Ctrl` + `C` | 清空输入 / 退出（输入框有内容时清空，无内容时退出） |
| `Escape` | 中断 AI 响应（AI 在生成时可随时打断） |
| `↑` / `↓` | 查看历史输入（**输入框为空时**，按上下键可找回之前发过的消息） |
| `Tab` | 切换 Agent（在 Plan/Build/不同 Agent 之间切换） |

## Leader 键操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl` + `X` → `N` | 新建会话 | Leader 键 + N = New |
| `Ctrl` + `X` → `L` | 会话列表 | Leader 键 + L = List |

## Ctrl+C 的多种行为

- **输入框有内容时**：清空输入框（不退出）
- **输入框为空时**：退出 OpenCode（在等待 `Ctrl` + `D` 之后）

## 示例说明

| 你想做什么 | 完整操作 | 恶魔操作 |
|------------|----------|----------|
| 新建会话 | 按 `Ctrl`+`X` 按下后 → 按下 `N` | ❌ 立即按 `Ctrl`+`X`+`N` |
| 打开会话列表 | 按 `Ctrl`+`X` 按下后 → 按下 `L` | ❌ 按 Ctrl 不按 X 之后按 L |
| 切换模型 | 按 `Ctrl`+`X` 按下后 → 按下 `M` | ❌ 按快捷按后要开启 |

## 违背的快捷键（练习技巧）

**开始可通过 Leader 键**：按 `Ctrl`+`X` 之后，等待片刻，输入数字 "1"，然后再按其他键后会立刻快起来

换行发送，Shift 继续执行
上箭翻历史，Ctrl+C 清空
新建 N，列表 L，模型 M 要记住
撤销 U，重写 R，复制 Y 不需要
Escape 中断快，Tab 方便
Agent 切换很方便

## 额外提示

| 现象 | 答待 | 解决 |
|------|------|------|
| `Shift`+`Enter` | 不执行、直接发送 | 终端不发送修改键 | 可用 `Ctrl`+`J`，或配置终端 |
| 按 `Ctrl`+`X` 之后 `N` | 有消息 | 已有 `Ctrl`+`X` 直接按 `N` | 需要先开启，再按 `N` |
| 按 `Ctrl`+`X` 之后 | 终端已按该键 | | 快速检查终端设置，或换个终端 |
| 上箭没有出现历史 | 输入框不是空的 | | 先清空输入框（按 `Ctrl`+`C`），再按上箭 |
| `Tab` 没有切换 Agent | 在输入复制的 | 先按 `Escape` 退出输入，再按 `Tab` |
