OpenCode 快捷键说明

快捷键 功能 快捷键 功能
Enter 发送消息/到达底部 Shift+Enter 不发送/多行输入
Ctrl+C 清空输入/退出 Escape 中断 AI 响应
↑/↓ 查看历史(空输入框) Tab 切换 Agent
Ctrl+X→N 新建会话 Ctrl+X→L 会话列表
Ctrl+X→M 模型标签 F2 快速切换模型
Ctrl+X→U 撤销消息 Ctrl+X→R 重写消息
Ctrl+P 命令面板 Ctrl+X→Y 复制 AI 回复
Ctrl+X→C 折叠上下文 Ctrl+X→B 切换侧边栏
PageUp/PageDown 翻页浏览 Ctrl+X→T 主题标签
Ctrl+A 跳到行首 Ctrl+E 跳到行尾
Ctrl+K 删除光标到行尾 Ctrl+U 删除光标到行首
Ctrl+W 删除上一个单词 Alt+B 后退一个单词
Alt+F 前进一个单词

操作说明

类型 说明
Ctrl+C 行为 输入框有内容时：清空；为空时：退出
Leader 键 按 Ctrl+X 后等待片刻再按键（N=新会话，L=列表，M=模型）
练习口诀 换行发送Shift，上箭翻历史，新建N列表L模型M，撤销U重写R复制Y，Escape中断Tab切换Agent

常见问题

现象 解决方法
Shift+Enter 直接发送 终端不发送修改键，可用 Ctrl+J 或配置终端
Ctrl+X→N 无反应 需先按 Ctrl+X 等待开启模式，再按 N
上箭无历史记录 先清空输入框（Ctrl+C）
Tab 无法切换 先按 Escape 退出输入模式

✅ 快速检查

  • Shift+Enter 不发送
  • ↑ 查看历史
  • Ctrl+C 清空
  • Ctrl+X→N 新会话
  • Ctrl+X→L 列表
  • Escape 中断
