## 六、调制方案（配方卡）⭐

> 以下配方按「新手友好 → 进阶 → 女士甜酒 → 无醇」分级，每张卡片可直接打印摆在吧台

### 🌿 清爽派（草坪首选，出片率最高）

#### 1. Aperol Spritz 意大利开胃酒 ⭐⭐⭐ 草坪爆款
- **配方**：Aperol 30ml + 普罗塞克（或起泡酒）60ml + 苏打水 30ml + 橙片
- **比例口诀**：3-2-1（起泡酒 3 份 / Aperol 2 份 / 苏打 1 份）
- **做法**：杯中加冰 → 倒 Aperol → 加起泡酒 → 加苏打水 → 橙片装饰
- **风味**：橙红通透、微苦回甘、气泡感
- **度数**：约 8°

#### 2. Gin Tonic 金汤力
- **配方**：金酒 45ml + 汤力水 120ml + 青柠角
- **做法**：杯中加冰 → 倒金酒 → 慢慢倒汤力水 → 挤青柠汁 + 投入
- **风味**：清透草本、清爽解腻
- **度数**：约 10°

#### 3. Mojito 莫吉托 ⭐ 宾客最爱
- **配方**：白朗姆 45ml + 青柠汁 20ml + 糖浆 20ml + 薄荷叶 8-10 片 + 苏打水
- **做法**：薄荷叶+糖浆在杯底轻捣 → 加朗姆+青柠汁 → 加碎冰 → 顶部加苏打水 → 搅匀
- **风味**：薄荷清凉、古巴风情
- **度数**：约 10°

### 🍑 女士甜酒派（小甜酒直接调）

#### 4. 螺丝刀 Screwdriver
- **配方**：伏特加 45ml + 橙汁 90ml
- **做法**：杯中加冰 → 倒伏特加 → 加橙汁 → 搅匀
- **风味**：果汁感强、无酒精感
- **度数**：约 14°

#### 5. 龙舌兰日出 Tequila Sunrise
- **配方**：龙舌兰 45ml + 橙汁 90ml + 红石榴糖浆 15ml
- **做法**：杯中加冰 → 龙舌兰+橙汁搅匀 → 沿杯壁缓慢倒入红石榴糖浆（沉底形成渐变）
- **风味**：渐变红色、果味香甜
- **度数**：约 12°

#### 6. 百利甜牛奶
- **配方**：百利甜 45ml + 冰牛奶 90ml
- **做法**：杯中加冰 → 倒百利甜 → 加牛奶 → 搅匀
- **风味**：奶茶感、酒精感弱
- **度数**：约 8°

### 🍸 进阶派（懂酒的宾客）

#### 7. 玛格丽特 Margarita
- **配方**：龙舌兰 45ml + 君度 20ml + 青柠汁 15ml + 盐边
- **做法**：杯口用青柠角擦湿 → 蘸盐边 → 所有材料加冰入调酒壶摇匀 → 滤入盐边杯
- **风味**：酸甜咸平衡、经典款
- **度数**：约 22°

#### 8. 古典 Old Fashioned
- **配方**：波本 60ml + 糖浆 5ml + 安高天娜苦精 2 滴 + 橙皮
- **做法**：杯中放糖浆+苦精 → 加冰块 → 倒波本 → 挤橙皮油喷杯口 → 搅匀
- **风味**：威士忌浓烈、男士向
- **度数**：约 32°

#### 9. 威士忌酸 Whiskey Sour
- **配方**：波本 60ml + 柠檬汁 25ml + 糖浆 15ml + 蛋清（可选）
- **做法**：所有材料入调酒壶干摇（无冰）→ 再加冰湿摇 → 滤入杯中
- **风味**：酸甜平衡、蛋清绵密
- **度数**：约 22°

### 🍷 烈酒派对款

#### 10. 长岛冰茶 Long Island Iced Tea（⚠️ 烈！）
- **配方**：伏特加/金酒/朗姆/龙舌兰/君度 各 15ml + 柠檬汁 25ml + 糖浆 20ml + 可乐加满
- **做法**：所有酒+柠檬汁+糖浆入调酒壶加冰摇匀 → 滤入杯 → 加可乐加满
- **风味**：看似冰茶实则 22°+，警告！
- **度数**：约 22°

### 🫧 无酒精 Mocktail（开车/孕妇/不喝酒）

#### 11. 无酒精莫吉托 Virgin Mojito
- **配方**：青柠汁 20ml + 糖浆 20ml + 薄荷叶 + 苏打水
- **做法**：同莫吉托，去掉朗姆酒

#### 12. 水果气泡饮
- **配方**：蔓越莓汁 60ml + 橙汁 60ml + 苏打水 + 水果装饰
- **做法**：杯中加冰 → 倒果汁 → 加苏打水 → 水果装饰

## 八、关键 Tips

1. **出片率 TOP3**：Aperol Spritz（橙红）、莫吉托（薄荷绿）、金汤力（清透）
2. **必出无醇区**：开车/孕妇/不喝酒的宾客必备，标注清楚
3. **配方卡按「刻度线」标**：让宾客按 1:2:3 自己倒，自助不乱
4. **啤酒冰桶 ×2**：一个放白啤/果啤，一个放精酿，视觉清爽
5. **避坑**：淘宝买酒有假酒风险，大品牌优先天猫旗舰/京东自营
6. **参考 @丁丁向前进**（545👍 / 689⭐）：自己画的 DIY 酒单物料清单，**最贴婚礼自助调酒场景**
