## 基因检测专项

### 基因检测有必要做吗？

**强烈建议做！**

| 理由 | 说明 |
|-----|------|
| 携带者无明显症状 | 很多遗传病携带者本人没有任何症状 |
| 遗传风险 | 夫妻双方都是携带者，孩子有25%概率患病 |
| 早期干预 | 早发现早干预，避免悲剧 |
| 免费政策 | 北京有免费政策（4种基础版），不做白不做 |

### 基因检测套餐对比

| 套餐名称 | 包含项目 | 费用 | 适合人群 |
|---------|---------|------|---------|
| **免费基础版** | SMA、耳聋、脆性X、地贫（4种） | **0元** | 所有符合免费条件的人 |
| **19种单基因** | 19种常见单基因遗传病 | 约800元 | 想更全面筛查 |
| **顺义妇幼女性** | 脆性X + 耳聋 + 地贫 + SMA + 更多 | 2600元 | 只做基因检测 |
| **顺义妇幼男性** | 耳聋 + 地贫 + SMA + 更多（无脆性X） | 1800元 | 只做基因检测 |

### 基因检测包含哪些病？

#### 免费基础版（4种）

1. **SMA**（脊髓性肌萎缩症）- 婴儿头号遗传杀手
2. **耳聋基因** - 先天性耳聋
3. **脆性X综合征** - 智力障碍遗传病（女性必查）
4. **地中海贫血** - 南方地区高发

#### 19种扩展版（常见病种）

- 苯丙酮尿症
- 肝豆状核变性
- 糖原贮积症
- 先天性肾上腺皮质增生症
- 囊性纤维化
- 血友病
- 杜氏肌营养不良
- ...等19种

### 基因检测建议

| 情况 | 建议 |
|-----|------|
| 普通备孕夫妻 | 先做免费版4种，有问题再扩展 |
| 有家族遗传病史 | 直接做19种或更全面的筛查 |
| 高龄备孕 | 直接做19种 |
| 有不良孕产史 | 直接做19种 + 染色体 |

09:30 - 检查

11:00 - 完成
