# 开封婚礼攻略

> 整理自小红书，涵盖开封室外婚礼场地推荐、筹备流程及婚宴酒店选定注意事项。

---

## 一、开封室外婚礼概览

开封有丰富的室外婚礼场地资源，代表性场景包括：

- **草坪婚礼**：铂尔曼酒店草坪、开元名都金明池畔草坪、海棠花语庭院
- **小院婚礼**：农村或民宿小院，自然感强，费用相对低
- **湖景草坪**：开元名都大酒店坐落于金明池原址，天然湖景做背景

---

## 二、室外婚礼筹备流程

### 1. 提前规划时间线

| 时间节点 | 事项 |
|---------|------|
| 婚礼前 8-12 个月 | 确定婚礼风格、预订婚礼团队 |
| 婚礼前 8-10 个月 | 预订场地/酒店（热门日期可能提前 1 年被订满） |
| 婚礼前 6 个月 | 预订婚纱、敬酒服、跟妆造型 |
| 婚礼前 3-6 个月 | 确定菜单、试菜、彩排 |
| 婚礼前 1-2 个月 | 定制伴手礼、喜糖、婚礼 Logo 等细节 |

### 2. 核心筹备项

**① 婚礼策划团队**
- 口碑团队最重要，避免只会忽悠的
- 需与设计师多轮沟通确认效果图
- 费用参考：包含摄像摄影约 **3 万元**（不含甜品台）

**② 场地选择**（详见第三部分）

**③ 婚纱礼服**
- 出门纱 + 主纱（鱼尾/蓬蓬款按喜好选）+ 敬酒服
- 本地定制婚纱下来约 **1 万+**
- 旗袍敬酒服定制约 **2000 元**

**④ 跟妆造型**
- 本地造型师价格约 **2000 元**起

**⑤ 婚戒**
- 按预算选择，注意留足时间刻字定制

### 3. 室外婚礼布置要点

**场地装饰参考（金明池风格）：**
- 背景：镂空木质框架 + 透明亚克力挂件，以湖景做天然背景
- 花艺：向日葵 + 玫瑰，色调阳光明亮
- 座椅：水晶竹节椅，椅背系气球
- 气氛点缀：池边/围栏装饰白色纱幔与向日葵

**功能区配置建议：**
- 甜品区：小甜点 + 饮品，以波西米亚屏风为背景，摆放新人合影
- 水吧区：鸡尾酒 + 饮用水，遮阳伞遮顶，爱迪生灯泡装饰（白天夜晚均可用）

### 4. 室外婚礼注意事项

- **天气应急方案**：提前准备雨棚或室内备用方案，开封四月仍较凉
- **时间控制**：室外仪式建议控制在 1 小时以内，避免宾客疲惫
- **灯光**：爱迪生灯泡、蜡烛等暖光源，夜间效果极佳
- **音响**：室外音响需比室内功率更大，提前测试
- **彩排**：必须提前到场地彩排，熟悉动线

## 三、开封室外婚礼场地推荐

### 1. 开封建业铂尔曼酒店
- **位置**：开封市区，紧邻古城湖景
- **特点**：草坪 + 古城湖景背景，法式高端仪式感，脚踩青草、头顶星光
- **价格参考**：餐标约 3699 元/桌起（可议）
- **亮点**：曾有新人作为首位草坪婚礼举办者，获赠两晚套房
- **适合人群**：追求高端仪式感、预算充足的新人

### 2. 开封开元名都大酒店（金明池畔）
- **位置**：金明池原址，历史文化底蕴深厚
- **特点**：五星酒店，室内 + 户外均可，户外效果更佳，早晨湖边拍照绝美
- **场地费**：名都厅 2000 元，开元厅 5000 元
- **餐标**：1800 元/桌起，包间 2000 元
- **亮点**：室内布置偏传统，但户外草坪自然感强

### 3. 海棠花语民宿酒店
- **位置**：市中心半亩园内
- **特点**：小众私密，园林景色极好，适合人数少的精致婚礼
- **场地费**：3500 元
- **餐标**：约 1400 元/桌（优惠后）
- **最大桌数**：15 桌（超出人数可在此办典礼，另找地方摆酒）
- **婚房**：2999 元（含露台，可开小型晚宴）
- **适合人群**：喜欢小众、精致风格，宾客人数 ≤ 150 人

## 四、婚宴酒店选定注意事项

### 1. 选址核心维度

| 维度 | 关注点 |
|------|--------|
| 场地大小 | 确认能容纳的最大桌数，1 桌约 10 人 |
| 停车 | 地下停车场容量、收费标准 |
| 交通 | 周边道路是否在修缮，嘉宾抵达是否方便 |
| 层高 | 室内厅层高影响婚庆布置，建议 ≥ 4.5 米 |
| 酒店配套 | 能否提供婚房、新娘房，是否配餐 |

### 2. 签合同前必问清单

**价格与桌数**
- [ ] 最低消费多少桌？免场地费的条件是什么（注意：部分酒店要桌数+消费额双达标）
- [ ] 是否可以加桌？加桌价格如何？
- [ ] 非大日子是否有折扣？

**菜品与服务**
- [ ] 餐标包含什么？烟酒是否可以自带，开瓶费多少？
- [ ] 是否需要试菜？试菜几道菜、几个人？
- [ ] 宴会当天菜品是否与试菜一致？

**场地与配套**
- [ ] 场地使用时间是几点到几点？超时如何收费？
- [ ] 是否提供迎宾区？
- [ ] 新娘化妆间/休息室在哪里？面积多大？是否有餐点提供？
- [ ] 婚礼当天停车如何安排？

**婚庆相关**
- [ ] 是否必须使用酒店指定婚庆公司？可以外请吗？
- [ ] 婚庆进场布置时间？
- [ ] 是否提供音响、灯光等基础设备？

**合同细节**
- [ ] 定金可否退？退定金条件是什么？
- [ ] 涨价条款：一年一个价，合同锁定后是否还会变动？
- [ ] 如遇突发情况（如疫情、停业）如何处理？

### 3. 谈判技巧

- **敢开口就赢一半**，不要不好意思谈
- **桌数多 = 话语权大**，人数越多越有砍价空间
- **非热门日期**（非节假日、非大日子）可以大胆压价
- **货比三家**：多看几家酒店再比价，同酒店不同时间问价可能不同
- **不要随便交钱**，签合同前务必确认所有条款

## 五、小预算室外婚礼参考

- 总预算 3 万元可完整举办草坪婚礼（含婚礼策划团队 + 摄影摄像）
- 节省技巧：
  - 甜品台自己制作（省 3000-5000 元）
  - 选择酒店刚开业时谈合作，往往有优惠
  - 非热门日期举办
  - 婚纱选本地定制而非知名品牌门店
