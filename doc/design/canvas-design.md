# canvas-design

创建视觉艺术作品（PNG 和 PDF 格式）的技能。

## 功能描述

当用户要求创建海报、艺术作品、设计或其他静态作品时激活。

## 流程

### 第一步: 设计哲学创建

创建**视觉哲学**（不是布局或模板），通过以下方式表达:
- 形式、空间、颜色、构图
- 图像、图形、形状、图案
- 作为视觉点缀的最少文字

#### 命名运动 (1-2 词)

"Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

#### 表达哲学 (4-6 段)

通过以下方式表达如何通过视觉表现哲学:
- 空间与形式
- 颜色与材质
- 规模与节奏
- 构图与平衡
- 视觉层次

#### 关键原则

- **避免冗余**: 每个设计方面提及一次
- **强调工艺**: 强调最终作品看起来像花费了大量时间
- **留出创意空间**: 保持足够具体但有诠释空间

### 第二步: 画布创建

使用设计哲学作为基础创建作品:

1. 使用设计哲学
2. 创建高度视觉化的单页 PDF 或 PNG 输出
3. 通常使用重复图案和完美形状
4. 将抽象哲学设计视为科学圣经
5. 使用有限的、有凝聚力的调色板
6. 文本始终最小化和视觉优先

### 文本处理

- 文本始终最小化和视觉优先
- 字体应该纤细
- 没有什么会从页面脱落或重叠
- 检查所有文本、图形和视觉元素有呼吸空间和清晰分隔

### 输出

生成:
1. 最终作品作为可下载的 .pdf 或 .png 文件
2. 作为 .md 文件的设计哲学

### 完善

如果需要完善:
- 避免添加更多图形
- 而是完善已创建的内容
- 使其极其清晰
- 完全尊重设计哲学和极简主义原则

## 示例设计哲学

**"Concrete Poetry"**
- 通过纪念性形式和大胆几何进行交流
- 大色块、雕塑式排版
- 波兰海报能量与勒柯布西耶相遇
- 文字作为稀有、强大的姿态
