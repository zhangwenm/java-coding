## 🔍 Apify 技能使用示例

### 1. apify-ultimate-scraper - 通用网页抓取

**场景 1：抓取 Google Maps 商家信息**
```
用户：抓取北京朝阳区所有咖啡厅的信息，包括名称、地址、电话、评分
→ 触发 apify-ultimate-scraper
```

**场景 2：采集 Instagram 帖子数据**
```
用户：采集 @username 最近 20 条帖子的点赞数、评论数、发布时间
→ 触发 apify-ultimate-scraper
```

**场景 3：多平台数据采集**
```
用户：从 Instagram、Facebook、Twitter 采集关于"新能源车"的话题讨论
→ 触发 apify-ultimate-scraper
```

## 📝 常用组合示例

### 组合 1：小红书内容运营工作流

```
1. "登录我的小红书账号" → xhs-auth
2. "搜索'科学备孕'热门笔记" → xhs-explore
3. "分析这些笔记的内容特点" → xhs-content-ops
4. "写一篇类似风格的笔记" → xhs-publish
5. "给竞品笔记评论引流" → xhs-interact
```

### 组合 2：竞品分析工作流

```
1. "抓取竞品在各大平台的数据" → apify-ultimate-scraper
2. "分析竞品社交媒体表现" → apify-content-analytics
3. "追踪竞品广告投放" → apify-competitor-intelligence
4. "生成竞品分析报告" → deep-research
5. "把报告写入飞书文档" → feishu-doc
```

### 组合 3：电商数据监控工作流

```
1. "监控竞品在亚马逊的价格" → apify-ecommerce
2. "追踪用户评论变化" → apify-brand-reputation-monitoring
3. "生成周报" → docx
4. "发送给团队" → feishu-doc
```
